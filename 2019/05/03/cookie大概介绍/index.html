cookie大概介绍

Cookie 2019/05/03

1、什么是Cookie
  Cookie是W3C组织提出的，最早由NetScape社区发展出的一种机制。
  Cookie是存储于访问者计算机中的变量，每当同一台计算机通过浏览器请求某个页面时，都会发送个Cookie。
  Cookie的作用就是用于解决“如何记录客户端的用户信息”
    1. 当用户访问web页面时，他的名字可以记录在Cookie中。
    2. 当用户下一次访问时，可以在Cookie中读取用户访问记录。
  Cookie实际上是一段文本信息（上限为4kb）。客户端请求服务器，如果服务端需要记录该用户状态，就使用response向客户端浏览器颁发一个Cookie。客户端浏览器可以把Cookie保存起来。当浏览器再请求该网站时，浏览器把请求的网址连同该Cookie一同提交给服务器。服务器检查该Cookie，以此来辨认用户状态。服务还可以根据需要修改Cookie的内容。

2、Cookie的属性

属性 功能
name cookie的名称,cookie一旦创建，名称便不可更改
value cookie的值
domain 通过该属性设置顶域及自己的域，不能设置其他域，以此实现Cookie跨域
path 设置可访问此Cookie的页面路径，“/”表示本域名下的路径都可以访问，最后一个字符必须为“/”
expires/Max-Age cookie超时时间，不设置话，默认值为session,意思是cookie和session会一起实效，浏览器关闭，cookie实效
size cookie的大小
secure 改Cookie是否仅被使用安全协议传输，默认为false
HttpOnly 只能使用http请求
comment cookie的用处说明，浏览器显示cookie信息时，显示该说明
version Cookie使用的版本号。0表示遵循Netscape的Cookie规范，1表示遵循W3C的RFC 2109规范
